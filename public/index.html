List of APIs

List of Currency Rates

API:
http://52.43.52.103:1234/list
METHOD TYPE:
GET
QUERY PARAMS:
source
DETAILS:
This api will get the list of currency rates for a given currency. By default, source currency will be INR & price will be 1 unit. You can change the source currency through the query param source
EXAMPLE:
URL: http://52.43.52.103:1234/list?source=USD 

{
  success: true,
  source: "USD",
  currency_values: [{
    name: "INR",
    value: 64.379997
  },
  ..]
}

Converstion Of Currency Rates

API:
http://52.43.52.103:1234/list/convert?source=CURRENCY_CODE&destination=CURRENCY_CODE
METHOD TYPE:
GET
QUERY PARAMS:
mandatory: source, destination; optional: price
DETAILS:
This api will convert the currency mentioned in the source query param to the currency mentioned in destination query param. Both source & destination query params are mandatory. By default price will be taken as 1 unit. You can change it by passing price query param which is optional
EXAMPLE:
URL: http://52.43.52.103:1234/list/convert?source=USD&destination=INR 

{
  success: true,
  source: "USD",
  destination: "INR",
  price: 1,
  converted_value: 64.379997
}

Note:
  • Each response from the server will be a JSON object
  • Each response will have a boolean field success, which is set to true in case of success
  • errors field will show the reasons for a unsuccessful call
